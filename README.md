# Photo-Gallery

A basic photo gallery, created using legacy content.

Some tecnologies used:

* JavaScript(Obviously);
* HTML;
* CSS;
* SASS/SCSS;
* JQuery;
* Webpack;
* Ajax;
* Node.

## Running the apps

```bash
# Install dependencies
$ npm i

# To run applications
$ npm start

# To use Webpack constructor
$ npm run build
```
